Computador Básico

1
Ligar o Computador

O primeiro passo para usar um computador é ligá-lo. Localize o botão de energia (geralmente tem este símbolo: ) e pressione-o.

Na simulação abaixo, você pode praticar clicando no botão "Ligar/Desligar".

2
Área de Trabalho

Após ligar o computador, você verá a área de trabalho (desktop). Nela estão os ícones de programas que você pode usar.

Clique nos ícones para abrir os programas. Experimente clicar nos ícones na simulação abaixo.

3
Usando o Mouse

O mouse é usado para apontar e selecionar itens na tela. Ele tem dois botões principais:

  • Botão esquerdo: Usado para clicar e selecionar
  • Botão direito: Normalmente abre um menu de opções

Pratique movendo o cursor e clicando nos elementos da simulação.

4
Desligar o Computador

Quando terminar de usar o computador, é importante desligá-lo corretamente. Clique no botão "Ligar/Desligar" novamente na simulação para ver como funciona.

Em um computador real, você geralmente clica no menu Iniciar e escolhe a opção "Desligar".

5
Teclado

O teclado é usado para digitar textos e números no computador. Ele tem várias teclas importantes:

  • Letras e números: Para digitar textos e valores
  • Enter: Confirma ações ou inicia uma nova linha
  • Backspace: Apaga o caractere à esquerda do cursor
  • Teclas de setas: Movem o cursor na tela

Experimente digitar algo em um editor de texto para praticar.

6
Menu Iniciar

O Menu Iniciar é onde você encontra todos os programas instalados no computador. Ele geralmente fica no canto inferior esquerdo da tela.

Para abrir o Menu Iniciar, clique no ícone do Windows (no canto inferior esquerdo) ou pressione a tecla Windows no teclado.

A partir do Menu Iniciar, você pode acessar programas, configurações e desligar o computador.

7
Janelas e Botões

Os programas no computador abrem em "janelas". Cada janela tem três botões importantes no canto superior direito:

  • Minimizar (−): Oculta a janela temporariamente
  • Maximizar (□): Expande a janela para ocupar toda a tela
  • Fechar (×): Fecha o programa

Você pode mover uma janela arrastando sua barra superior e redimensioná-la puxando suas bordas.

8
Arquivos e Pastas

Os arquivos são documentos, fotos, vídeos ou outros conteúdos guardados no computador. As pastas são como gavetas que organizam esses arquivos.

Para abrir o explorador de arquivos, clique no ícone da pasta na barra de tarefas ou no menu Iniciar.

Dentro do explorador de arquivos, você pode:

  • Abrir arquivos com um duplo clique
  • Criar novas pastas com o botão direito → Nova pasta
  • Copiar e colar usando os botões direitos ou Ctrl+C e Ctrl+V
9
Conexão com a Internet

Para usar a internet, seu computador precisa estar conectado a uma rede Wi-Fi ou cabo de rede.

Para se conectar a uma rede Wi-Fi:

  1. Clique no ícone de Wi-Fi na barra de tarefas (canto inferior direito)
  2. Selecione sua rede na lista
  3. Digite a senha se for solicitada
  4. Clique em "Conectar"

Quando conectado, o ícone de Wi-Fi mostrará barras indicando a força do sinal.

10
Programa Antivírus

Um antivírus protege seu computador contra programas maliciosos que podem danificar arquivos ou roubar informações.

A maioria dos computadores já vem com um antivírus instalado (como o Windows Defender). É importante:

  • Manter o antivírus sempre atualizado
  • Realizar verificações periódicas do sistema
  • Não abrir anexos de e-mails desconhecidos
  • Ter cuidado ao baixar arquivos da internet
11
Atualizações do Sistema

Atualizações são melhorias e correções de segurança para o sistema operacional do seu computador.

Para verificar atualizações no Windows:

  1. Abra o menu Iniciar e clique em "Configurações" (ícone de engrenagem)
  2. Selecione "Atualização e Segurança"
  3. Clique em "Verificar por atualizações"

É recomendado manter seu sistema sempre atualizado para maior segurança e desempenho.

12
Impressora e Dispositivos

Você pode conectar vários dispositivos ao seu computador, como impressoras, fones de ouvido, pendrives, etc.

Para configurar uma impressora:

  1. Conecte a impressora ao computador e ligue-a
  2. Abra as Configurações e selecione "Dispositivos"
  3. Clique em "Impressoras e scanners"
  4. Selecione "Adicionar impressora" e siga as instruções

Pendrives e outros dispositivos USB geralmente são reconhecidos automaticamente quando conectados.

O que é um computador?

Um computador é uma máquina que processa informações. Ele tem alguns componentes principais:

  • Monitor: É a tela onde você vê as informações.
  • CPU (Unidade Central de Processamento): É o "cérebro" do computador.
  • Teclado: Usado para digitar texto e números.
  • Mouse: Usado para apontar e clicar na tela.

Experimente ligar o computador virtual acima e clique nos ícones para abrir programas!

Bem-vindo à Internet!

Navegando na Internet com Segurança

A internet é uma ferramenta poderosa, mas requer alguns cuidados:

  • Verifique se os sites são seguros (símbolo de cadeado).
  • Nunca compartilhe senhas ou dados bancários.
  • Tenha cuidado com e-mails de remetentes desconhecidos.
  • Use senhas fortes e diferentes para cada site.

Experimente usar a barra de pesquisa acima para simular uma busca na internet!

O que é programação?

Programação é como dar instruções para um computador realizar tarefas. É como uma receita de bolo, onde cada passo deve ser seguido na ordem correta.

Conceitos básicos:

  • Variáveis: São como caixas que guardam informações.
  • Funções: São conjuntos de instruções que realizam uma tarefa.
  • Condicionais: São decisões baseadas em condições (se isso, então aquilo).

Glossário de Termos Técnicos

Confira abaixo os principais termos técnicos relacionados à informática e suas definições.

Hardware

São os componentes físicos do computador, ou seja, as peças e equipamentos que podem ser tocados, como monitor, teclado, mouse, etc.

Exemplos: CPU, placa-mãe, disco rígido, memória RAM.

Software

São os programas e sistemas que fazem o computador funcionar. São as instruções que dizem ao hardware o que fazer.

Exemplos: Sistema operacional (Windows, MacOS), aplicativos (Word, Excel), navegadores de internet.

Internet

Rede global que conecta milhões de computadores em todo o mundo, permitindo a troca de informações e o acesso a diversos serviços.

Você usa a internet quando acessa sites, envia e-mails ou participa de redes sociais.

Wi-Fi

Tecnologia que permite a conexão sem fio (wireless) de dispositivos à internet, dispensando o uso de cabos.

Quando você conecta seu celular ou notebook à internet de casa sem usar cabos, está usando Wi-Fi.

Navegador

Programa utilizado para visualizar páginas na internet, como sites e blogs.

Exemplos: Google Chrome, Mozilla Firefox, Microsoft Edge, Safari.

URL

Endereço que identifica um site na internet, geralmente começando com "http://" ou "https://".

Exemplo: https://www.uniateneu.edu.br

Download

Processo de transferir (baixar) um arquivo da internet para o seu computador.

Quando você salva uma foto ou documento da internet no seu computador, está fazendo um download.

Upload

Processo de transferir um arquivo do seu computador para a internet.

Quando você envia uma foto para as redes sociais ou anexa um documento em um e-mail, está fazendo um upload.

Vírus

Programa malicioso que infecta o computador, podendo danificar arquivos, roubar informações ou prejudicar o funcionamento do sistema.

Um vírus pode chegar ao seu computador através de e-mails suspeitos, sites duvidosos ou pendrives infectados.

Antivírus

Programa que protege o computador contra vírus e outras ameaças digitais.

Exemplos: Avast, AVG, Norton, Kaspersky.

Programação

Arte de criar instruções para que o computador execute tarefas específicas, utilizando linguagens de programação.

JavaScript, Python e Java são exemplos de linguagens de programação.

Algoritmo

Sequência de passos lógicos para resolver um problema ou executar uma tarefa.

Uma receita de bolo é um exemplo de algoritmo do dia a dia, pois contém uma sequência de passos a seguir.

Metodologia de Ensino

Abordagem Metodológica

A metodologia utilizada neste curso baseia-se na aprendizagem ativa, onde os participantes são incentivados a praticar enquanto aprendem. Considerando o público-alvo (pessoas acima de 60 anos), recomenda-se:

  • Ritmo adequado: Conduzir as explicações com calma, respeitando o tempo de aprendizado de cada um.
  • Exemplos práticos: Utilizar exemplos que façam sentido no cotidiano dos participantes.
  • Repetição construtiva: Reforçar conceitos importantes de forma variada.
  • Paciência e empatia: Estar preparado para repetir instruções e evitar termos técnicos complexos.

Avaliação do Aprendizado

Métodos de Avaliação

A avaliação não deve ser um fator de stress para os participantes. Recomenda-se uma abordagem formativa, focada no processo de aprendizagem:

  • Observação direta: Acompanhar o desenvolvimento durante as atividades práticas.
  • Autoavaliação: Pedir que os participantes reflitam sobre seu próprio progresso.
  • Exercícios práticos: Propor tarefas simples que apliquem o conhecimento adquirido.
  • Feedback construtivo: Fornecer retorno individualizado, destacando avanços e áreas para melhoria.
